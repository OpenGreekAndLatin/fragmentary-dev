Welcome to fragmentary-dev!

About

This repository contains EpiDoc XML versions of Ancient Greek and Latin fragmentary works/authors/texts.

These XML files were produced by data entry company Digital Divide Data who OCR'd, corrected and encoded the text in accordance with the latest EpiDoc standards. While the initial results are adequate, more can and should be done to improve the XML.
