Welcome to fragmentary-dev!

About

This repository contains EpiDoc XML versions of Ancient Greek and Latin fragmentary works/authors/texts. This work is part of the Leipzig Open Fragmentary Texts Series (LOFTS) at the University of Leipzig.

These XML files were produced by data entry company Digital Divide Data (DDD), who corrected and encoded hOCR output generated by Bruce Robertson in accordance with the latest EpiDoc standards. While the initial results are adequate, more can and should be done to improve the XML.
